jReply RGBA Color Picker Plugin for jQuery UI

Despite the fact that modern browsers have such good support for RGBA color assignments the number of usable and configurable RGBA color pickers are rather thin on the ground. This color picker is intended to help plug that gap. It is easy to configure, has a simple point and click user interface and offers the ability to store up to 8 named colors for future use. It has been tested in Chrome, Firefox, Safari for Windows, Opera and IE and should work in all recent versions of these browsers.

Usage

Typical usage would be as follows

  1. Add a suitably named button; e.g., "Choose Color"; where required in your application
  2. Bind to the click event for that button
  3. In the click event handler use the code 
     function cbk(rgba)
 {
  /*
   * the callback function gets the chosen color
   * as the string rgba(R,G,B,A)
   * typically you would use this string to update
   * the color/background color for a page element
   * in your application
  */
 }
 
 var options = {caption:"My RGBA Color Picker"rgba:"R,G,B,A"callback:cbk};
 $('<div>').jreply_rgbapicker(options);
